    • UI/UX Design


    UI/UX Design focuses on creating visually appealing and user-friendly digital
    experiences. It combines user interface design, which deals with the layout, colors,
    and interactive elements, with user experience design, which emphasizes usability, functionality, and the overall satisfaction of the user journey.


    What you will learn


    User Research and Personas

    Understanding target audiences through research, creating personas, and analyzing user needs to inform design decisions.


    Wireframing and Prototyping

    Developing low-fidelity wireframes and interactive prototypes to plan and test the layout and functionality of digital products.


    Visual Design Principles

    Learning about color theory, typography, spacing, and other design elements to create visually appealing interfaces.


    Interaction Design

    Designing interactive elements like buttons, animations, and transitions to enhance user engagement and usability.


    Usability Testing and Feedback

    Conducting usability tests to gather user feedback, identify pain points, and make iterative improvements to the design.


    Design Tools and Software

    Mastering industry-standard design tools such as Figma, Adobe XD, or Sketch for creating and refining UI/UX designs.
